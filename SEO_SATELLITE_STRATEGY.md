# SEO Satellite Site Strategy: lyvarajewels.shop

Use `lyvarajewels.shop` as a satellite SEO site targeting long-tail keywords with 50-100 blog posts that link back to `lyvarajewels.com`. This drives organic traffic and affiliate revenue without competing with main domain.

## Phase 3: Internal Linking Strategy

### Link Structure (Every Post Links Back to Main Domain)

**Type 1: Product Links** (High Priority)
- Link to specific products on main site
- Example: "Check out our collection of [product category]"
- Target: 2-3 internal links per post

**Type 2: Category Links** (Medium Priority)
- Link to main site category pages
- Example: "Browse all gold necklaces"
- Target: 1-2 category links per post

**Type 3: Related Content Links** (Low Priority)
- Link to related blog posts on main site
- Example: "Learn more about gold jewelry care"
- Target: 1 related content link per post

### Linking Anchor Text Examples
- "Shop gold jewelry for women"
- "Browse our gold necklace collection"
- "View all luxury gold jewelry"
- "Explore affordable gold rings"
- "See our best-selling gold bracelets"
- "Check out our gift-worthy gold jewelry"

## Phase 4: Content Generation Plan

### Automation Strategy
1. **Weekly Content Generation** (5 posts/week)
   - Use AI to generate 500-800 word posts
   - Each post targets 1 long-tail keyword
   - Include 2-3 internal links to main domain
   - Add hero image and metadata

2. **SEO Optimization**
   - Keyword in title, H1, first 100 words
   - Meta description with CTA
   - Internal links with relevant anchor text
   - Image alt text with keywords

3. **Publishing Schedule**
   - Monday-Friday: 1 post per day
   - Total: 50-100 posts in 10-20 weeks
   - Then: 1-2 posts per week for maintenance

## Phase 5: Domain Configuration

### DNS & Hosting Setup
1. Point `lyvarajewels.shop` to separate instance or subdomain
2. Configure as independent site (not redirect)
3. Set up separate analytics tracking
4. Create robots.txt to allow indexing

### Technical SEO
- Unique content (not duplicate of main site)
- Proper canonical tags pointing to main domain where appropriate
- XML sitemap for .shop domain
- Mobile-responsive design
- Fast loading speed

## Phase 6: Analytics & Tracking

### Key Metrics to Monitor
1. **Traffic Metrics**
   - Monthly unique visitors to .shop
   - Click-through rate to main domain
   - Bounce rate (should be low if links are working)

2. **Conversion Metrics**
   - Affiliate clicks from .shop → main domain
   - Affiliate conversions (sales)
   - Revenue generated from .shop traffic

3. **SEO Metrics**
   - Keyword rankings for target keywords
   - Organic traffic growth
   - Backlink profile

### UTM Tracking
Add UTM parameters to all internal links:
```
https://lyvarajewels.com/product/[id]?utm_source=shop&utm_medium=blog&utm_campaign=[keyword]
```
